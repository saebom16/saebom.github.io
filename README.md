# 포트폴리오 파일 구조

```
portfolio/
├── index.html          ← 메인 HTML (열어서 바로 실행)
├── css/
│   └── style.css       ← 모든 스타일 (라이트박스 포함)
├── js/
│   └── main.js         ← 모달 열기/닫기 + 이미지 슬라이더
└── resource/           ← 이미지 파일 모두 여기에
    ├── 프로필.jpg
    ├── p1_main.png ~ p6_main.png   (프로젝트 카드 썸네일)
    ├── P1_1.png ~ P1_5.png         (GDN 갤러리)
    ├── P2_1.png ~ P2_5.png         (모이닷 갤러리)
    ├── P3_1.png ~ P3_5.png         (Pairy 갤러리)
    ├── P4_1.png ~ P4_5.png         (SKT 갤러리)
    ├── P5_1.png ~ P5_7.png         (POOMY 갤러리)
    └── P6_1.png ~ P6_5.png         (키도미도 갤러리)
```

## 이미지 슬라이더 사용법
- 모달 안의 이미지를 클릭하면 라이트박스가 열립니다
- `‹` `›` 버튼 또는 키보드 ← → 로 이전/다음 이미지 이동
- `Esc` 또는 ✕ 버튼으로 닫기
- 배경 클릭으로도 닫기 가능
